- může být integrovaná na základní desce nebo dedikovaná
- Dělí se na:
	- grafický procesor
		- stará se o grafické výpočty
		- je to mozek GPU
		- stará se o komunikaci s grafickou pamětí
	- grafická pamět (VRAM)
		- úložiště pro zpracované data

# Výstupy
- vga - dnes se už moc
- HDMI
- Display port

# Chlazení
- aktivní
	- musí být napájené aby chladili
	- např. větrák
- pasivní
	- nepotřebuje napájení
